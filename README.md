# CloudNS Tools Site

Static landing page for CloudNS Tools.

## Local Preview

Open `index.html` directly in a browser, or serve the directory with any static file server.

## GitHub Pages

This repository is intended to publish from the default branch root.

Custom domain:

```text
cloudns.urbansheep.com
```

DNS:

```text
cloudns.urbansheep.com CNAME urbansheep.github.io
```
